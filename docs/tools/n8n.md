## 1. 它是干嘛的？

n8n 是一个开源的 **工作流自动化工具（workflow automation）**。

可以把它当作一条「自动化生产线」：

- 每个节点（Node）是一小步操作：收数据 / 调用 AI / 存数据库 / 发消息……
- 你用拖拽 + 连线的方式，把这些小步骤连成一条完整流程。
- 支持连接各种服务（Email、Google、Notion、数据库、OpenAI/Gemini 等），也可以直接调任何 HTTP API。

适合用来：

- 自动处理「重复性的信息工作」：收集 → 清洗 → 总结 → 存档 → 通知
- 搭建「多步 AI Agent 流程」：AI 理解任务 → 决定要调用哪些工具 → n8n 负责任务执行
- 将个人/小团队的各种工具“接到一起”
